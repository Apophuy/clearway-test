# ClearwayTest

## Старт сервера разработки:

```bash
ng serve
```

Сервер откроется по адресу `http://localhost:4200/`.

## Сборка проекта

```bash
ng build
```

Проект будет собран в `dist/` директории.

## Версия Node

Указана в файле .nvmrc <br/>
На проекте используется npm и nvm<br/>

## Используемые библиотеки:

- Angular
- Angular Material
